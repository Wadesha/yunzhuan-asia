# 职业资格考试刷题站 · 题库规模与题型调研报告

> 目的：在「盲目把每题型堆到 1000 道」之前，先把 **每个考试到底考几门、有哪些题型、单次考多少题、真实题库量级多大** 摸清楚，再决定合理的练习量目标。
> 数据来源：司法部 / 人社部 / 财政部 / 卫健委 / 教育部等官方公告，以及东奥、中华会计网校、233、华图等公开考情整理（2025–2026 考季）。
> 注：本仓库题库为「演示题」，非官方真题；下表「真实题库量级」指市面真题+模拟的总和规模，用于判断合理练习量，**不是我们要抄的数量**。

## 二、关键发现

1. **几乎所有考试都是「多门 + 客观为主、主观为辅」**。客观题（单/多/判/不定项）占每次考试的 70%–100%；主观题（简答/案例/论述/写作/翻译/作图）只在部分科目出现，且占比 30%–70% 不等。
2. **真实题库量级巨大**：法考/一建/二建/注会/公务员/教资/护士/医师/造价/监理/消防/安全/税务师/经济师/药师/软考等主流考试，市面真题+模拟都在 **数万级**；即便小类考试也有数千。
3. **当前 bank 严重偏薄**：48 个 bank 总题量 2536，平均每考试仅 ~53 题，而单次考试客观题就普遍 100–200 题。**法考仅 26 题、一建 60、二建 38** —— 连一次完整客观卷都凑不出，确实「远远不够」。
4. **题型差异大**：护士/医师是纯单选（A1–A4/B1）；初级会计含判断+不定项；一建/造价/监理/消防/安全/法考主观部分是案例；公务员申论/教资写作/注会简答是纯主观。

## 三、「每个题型 1000 道」有必要吗？—— 分析

**结论：没有必要，也不现实。**

- 若真按「每题型 1000」：以法考为例 = 单选 1000 + 多选/不定项 1000 + 主观 1000 ≈ **3000 题/考试**；公务员行测 5 模块 ×1000 = **5000**；48 个考试简单加权后总量会冲到 **10 万–30 万题**。
- 这会带来三个问题：
  1. **质量崩塌**：本仓库是合成/演示题，非官方真题。万级规模下只能靠模板化生成，答案正确性、干扰项区分度、解析准确性会显著下降，反而害了刷题体验。
  2. **体量失控**：即便只生成客观题，10 万+ 题的 JSON 体积、构建/校验耗时、CDN 负担都不可忽略。
  3. **收益递减**：刷题的核心是「覆盖考点 + 间隔重复」，单考点 1000 道重复收益极低；一次完整卷 100–200 题 + 章节练习 200–400 题已足够形成有效训练闭环。

**更合理的目标（建议）**：
- **主流考试（A/B/C/D 大类核心）**：客观题 **400–800 题/考试**，按分科拆（如法考客观 200 题量 → 题库 600–800 覆盖两卷各科目；一建 4 科 × 150 ≈ 600）。
- **中类考试（银行/证券/基金/期货/评估/审计/建筑师/测绘等）**：**200–400 题/考试**。
- **小类技能（播音/记者/出版/翻译/导游/验船/社工/软考/环评/计量等）**：**100–200 题/考试**（多数为单科）。
- **合计约 2.5 万–3.5 万道客观题**，是「够用且质量可控」的区间。

## 四、必须正视的引擎限制：主观题怎么办？

当前 `app.js` 判分引擎**只支持客观题**（单/多/判/不定项，点选即判分+朗读答案）。而下表这些考试含大量主观题，**无法自动判分**：

- 法考主观题（案例/论述/文书）
- 一建/二建/造价/监理/消防/安全/规划/房地产 的「案例分析」
- 公务员「申论」、教资「写作/简答/材料」
- 注会「简答/综合」、税务师「涉税实务简答」、CPA 计算
- 翻译 CATTI 实务、建筑师作图、播音口试

**三条路（需你拍板）**：
- **方案 X（只练客观）**：主观题科目只放客观部分，主观题型不做。最快、最稳。
- **方案 Y（主观转「例题+参考答案」卡片）**：主观题以「题目 + 展开参考答案」形式展示，不计分、可收藏，作为阅读/背诵材料。工作量中等，价值高。
- **方案 Z（扩展引擎支持主观）**：给引擎加「简答/论述 文本作答 + 关键词命中判分 / 自评」，工程量大，超出本次范围。
